What Is Random Number Guessing Game ?

I will explain it in context of computer and a person.

  • In this game, two players are required.
  • The first player(computer) generates a random number within a given range and the other player(person/user) has to guess that number.
  • If the guessed number doesn"t match with the random number then the player is given various hints i.e the guess is too low, try a higher number or guess is too high, try a lower number etc.
  • The player will keep guessing numbers until he/she finds the correct random number, while still being given hints.
  • Eventually, the user guesses the correct number, and this way the game will end.

It is fun and easy game to play , so have fun while doing it and goodluck!.
